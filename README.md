# PhD Thesis: Data-Efficient AI-Guided Energy- and Thermal-Aware Scheduling on Heterogeneous Multicore Systems

**Author:** Mohammad Pivezhandi
**Institution:** Wayne State University
**Defense Date:** December 18, 2025

## Overview

This dissertation presents data-efficient AI approaches for energy and thermal-aware scheduling on heterogeneous multicore embedded systems. The research addresses the challenge of deploying machine learning-based schedulers that require minimal training data while achieving near-optimal performance.
