# PhD Thesis: Data-Efficient AI-Guided Energy- and Thermal-Aware Scheduling on Heterogeneous Multicore Systems

**Author:** Mohammad Pivezhandi
**Institution:** University of Texas at El Paso (UTEP)
**Department:** Computer Science
**Defense Date:** December 18, 2025

## Overview

This repository contains the compiled PDF of my PhD dissertation on data-efficient AI-guided scheduling for heterogeneous multicore systems. The thesis explores machine learning approaches for energy and thermal-aware task scheduling.
